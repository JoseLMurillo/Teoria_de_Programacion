# Funciones

Las funciones son bloques de código reutilizables que realizan una tarea específica y pueden devolver un valor.

- **Ejemplo de Función:**
    ```
    función sumar(a, b)
        retornar a + b
    fin_función
    
    variable resultado = sumar(3, 4)
    imprimir "Resultado: " + resultado
    
    ```
